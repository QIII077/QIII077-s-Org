
# 轻享健康 (Light Health) 🥗

一款基于 **Google Gemini API** 构建的智能营养管理平台，采用极简的 **磨砂玻璃 (Glassmorphism)** 设计风格，专为追求生活品质的现代女性打造。

## ✨ 核心特性

- 🤖 **AI 魔法助手**：内置极简风格的 AI 对话助手，不仅提供专业的营养建议，还支持 AI 魔法照片编辑。
- 📸 **智能食物识别**：采用 Gemini 3 Flash 模型，支持通过拍照或搜索快速分析食物热量，告别繁琐记录。
- 🧪 **极简 UI 设计**：全站基于 Tailwind CSS 构建，拥有丝滑的磨砂玻璃质感、灵动的背景气泡与流体布局。
- 🔐 **登录系统**：内置模拟登录流程，保障个人信息的私密性与仪式感。
- 📊 **历史趋势**：可视化展示过去 7 天的热量摄入波动，科学管理健康目标。
- ⚡ **极致体验**：权限申请延迟至用户点击拍照时，拒绝初始弹窗骚扰。

## 🛠️ 技术栈

- **前端**: React 19 + TypeScript
- **样式**: Tailwind CSS (磨砂玻璃设计规范)
- **大模型**: Google Gemini API (Gemini 3 Pro/Flash, Gemini 2.5 Flash Image)
- **图标**: Lucide React
- **图表**: Recharts

## 🚀 快速开始

1. **环境准备**: 确保已获取 [Google AI Studio API Key](https://aistudio.google.com/)。
2. **运行环境**: 本项目通过 ES Modules 直接运行，无需复杂的构建工具。
3. **配置 API Key**: 将您的 API Key 设置在环境变量 `process.env.API_KEY` 中即可。

## 📸 界面预览

- **首页**: 今日热量进度条与 AI 个性化洞察。
- **记录**: 支持拍照、搜索、快速选择三种模式。
- **小助**: 亲切的女性挚友语气，提供 100 字以内的精炼回复。
